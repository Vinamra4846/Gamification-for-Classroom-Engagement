Memory Puzzle Game

In this game we will create 3 levels: easy, medium and hard. At the start of the game the shapes are hidden. The player clicks on the tiles to uncover the tile and when two similar tiles are uncovered the score increases. To complete the game the user has to uncover all the pairs of shapes. After completion of the game, the number of moves required to complete the game is also displayed on the screen.

Keyboard Jump

Keyboard jump game is a speed typing game that helps in improving the typing speed of players The object of Keyboard Jump Game Python Project is to build a keyboard jump game that helps players to increase their typing speed. We use pygame, random, and time modules in this project. In this project, the player has to press the same keys to the letters displayed on the game screen. If the player made an error while typing then the game gets over.
